``` 
VS Code Extensions:

```

# Install Angular Cli
```
npm install -g @angular/cli@16
```

# Create Angular app
```
ng new client
```

# Run Angular app
``` 
ng serve
```

# Install Angular Bootstrap (in Client directory)
```
ng add ngx-bootstrap
```
# Generate ssl cert
```
Instal mkcert
run command: mkcert -install
```

# Stop Hot Reload
```
dotnet watch --no-hot-reload  
```
